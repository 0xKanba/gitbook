# 🧭 جولة في لوحة التحكم (شجرة كائنات الموقع)

عندما تفتح الموقع، ستجد المكونات مرتبة بدقة كأنها قطع مكعبات ليغو (Lego). إليك هيكل شجرة كائنات الموقع المبسط لتفهمه بنظرة واحدة:

```
[الشاشة الرئيسية للموقع]
  ├── 🌐 البوابة العلوية (Header): تريك شعار HLsimple وحالة اتصال خزنتك الرقمية.
  ├── 📊 اللوحة المركزية (Main Terminal):
  │     ├── 📋 قائمة الأبطال السحرية (Sidebar): تختار منها الشيء الذي تريد تداوله.
  │     ├── 📈 الشاشة التفاعلية (Chart): خط يتحرك صعوداً وهبوطاً كالأفعوانية يريك الأسعار الحالية.
  │     └── ⚡ أزرار الأوامر (Order Panel): الأزرار السحرية لفتح وإغلاق الصفقات.
  └── 💰 قبو الأسرار السفلي (Footer): يريك صفقاتك الحالية، ورصيد أموالك المتبقية.
```
